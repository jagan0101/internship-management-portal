# Internship Management Portal

## Description
This project is an Internship Management Portal developed using Bolt.new.

## Features
- Student Registration
- Internship Application Management
- Dashboard
- Admin Panel
- Supabase Database Integration

## Technologies Used
- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase

## Installation

```bash
npm install
npm run dev
```
